# 在庫監視ツール v2.1 - ビックカメラ対応完了

## 概要
ビックカメラのBot対策（HTTP/2エラー）を回避するため、ブラウザ切り替えロジックを実装し、Firefoxを使用するように変更しました。

## 実装内容
1. **Firefox対応**: ビックカメラへのアクセス時のみFirefoxを使用するよう`monitor.py`を更新
2. **自動切り替え**: 設定ファイル（`config.yaml`）のサイト情報に基づき、Chromium/Firefoxを自動選択
3. **ワークフロー更新**: GitHub ActionsでFirefoxをインストールするように変更

ローカル環境で実行する場合は、Firefoxのインストールが必要です：

```bash

```
